添付されたPDF文書には、法的通知、テーブル、セクションの見出し、多段組文章などが含まれています。しかし、OCR処理によりノイズや読取不能な部分が混入している可能性があります。以下の手順に従って、文書を解析・抽出してください。

1. 文書全体を意味のある論理的なセクションに分割してください。各セクションは、見出しや区切り（改段落）に基づいて識別してください。

2. 各セクションから、以下の情報を抽出します。なお、正しく意味が取れる日本語の文章が抽出できなかった場合は、その項目はnullとしてください。また、明らかにOCRのノイズと思われる部分は除外してください。
   - "section_id": セクションごとのユニークな識別子（例："section_1"）
   - "page_number": セクションが開始するページ番号（識別できなければnull）
   - "title": セクションのタイトルまたは見出し（存在しなければnull）
   - "text": セクションの主要な本文。OCRノイズや読取不能な文字列は取り除き、意味のある正確な日本語の文章を出力してください。正しく抽出できない場合はnullとする。
   - "tables": セクション内にテーブルが存在する場合、各テーブルは以下の情報を含むオブジェクトとして抽出します：
       - "headers": テーブルの列見出し（文字列の配列）
       - "rows": テーブルの各行（各行は文字列の配列）
       - "notes": テーブルのデータ品質に関する注意事項、またはnull
   - "charts": セクション内にチャート（グラフ）が存在する場合、各チャートについて以下を抽出します：
       - "chart_type": "bar graph", "line graph", "pie chart" または "unreadable" のいずれか
       - "description": チャートの簡潔な説明
       - "data": チャートの主要データや要約（抽出できなければnull）
   - "images": セクション内に画像が含まれる場合、各画像について以下を抽出します：
       - "description": 画像の内容を説明するテキスト（分からなければnull）
       - "page_number": 画像が含まれるページ番号（識別できなければnull）

3. 上記の情報を以下のJSONスキーマに厳密に従って、余分なテキストや解説を一切含めず、**有効なJSONオブジェクトのみ出力**してください。

JSONスキーマの例:
{
  "sections": [
    {
      "section_id": "<ユニークな識別子>",
      "page_number": <整数またはnull>,
      "title": "<セクションタイトルまたはnull>",
      "text": "<意味のある正しい日本語の文章、またはnull>",
      "tables": [
        {
          "headers": ["<見出し1>", "<見出し2>", "..."],
          "rows": [["<行1の列1>", "<行1の列2>", "..."], ["<行2の列1>", "..."]],
          "notes": "<テーブルの注意事項またはnull>"
        }
      ],
      "charts": [
        {
          "chart_type": "<bar graph, line graph, pie chart, または 'unreadable'>",
          "description": "<チャートの説明>",
          "data": "<データの要点またはnull>"
        }
      ],
      "images": [
        {
          "description": "<画像説明またはnull>",
          "page_number": <整数またはnull>
        }
      ]
    }
  ]
}

注意:
- 出力は純粋に上記のJSONオブジェクトのみとしてください。その他の解説や余計なテキストは一切含めないこと。
- OCRによる読取不能な部分やノイズが多い場合は、その部分を抽出せずnullとしてください。

以上の指示に従い、結果を有効なJSONとして出力してください。
